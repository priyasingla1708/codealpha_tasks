# Task 2 - Image Gallery

This project is an interactive image gallery built with **HTML**, **CSS**, and **JavaScript**.

## ✨ Features
- Displays multiple images in a clean layout
- Hover effects for styling
- Navigation to browse through images

## ▶️ How to Run
1. Open `index.html` in your browser.
2. Hover over and click images to explore the gallery.
